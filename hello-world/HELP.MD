# About Gradle

Build lifecycle phases: 
- initialization
- configuration
- execution

`gradle tasks --all` - see full list of tasks
`gradle hello --dry-run` - check the tasks without executing
`gradle createArchive --console=verbose` - to see the outcomes of tasks
